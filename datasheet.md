# Datasheet for Dataset## Synthetic Single-Cell Dataset for Modeling Mitochondrial Permeability Transition Pore Behavior## MotivationThis dataset was created to support proof-of-concept computational analysis of mitochondrial permeability transition pore behavior across disease-relevant mitochondrial stress states. The purpose is to provide a structured synthetic dataset that can be used for descriptive analysis, perturbation comparison, classification, regression, and counterfactual modeling.The dataset was generated because experimentally calibrated, state-labeled, multi-variable mitochondrial datasets covering cardiac injury and cancer-associated adaptation in a unified format are difficult to obtain. A synthetic dataset allows controlled hypothesis testing and pipeline development before wet-lab validation.## CompositionEach row in the dataset represents one simulated single-cell observation. The dataset is complete by design, with no missing values unless the user later modifies the data.The dataset includes:- Biological state labels- Perturbation condition labels- Upstream mitochondrial stress variables- Intermediate pore-related variables- Downstream injury and survival-related outcomes### Example variable groups#### Categorical variables- state- perturbation#### Upstream continuous variables- dpsi_m or equivalent variable for mitochondrial membrane potential- matrix_ca- mtros- cypd_activity- ant_function- atpsyn_pore_propensity#### Intermediate variables- transient_pore_score- sustained_open_prob- sustained_open_status#### Downstream variables- mtDNA_release- viability_fraction- ATP_indexVariable names may differ slightly depending on implementation in the notebook.## Collection ProcessThis dataset was not collected from human participants, animal experiments, or laboratory instruments. It was generated computationally using state-dependent probability distributions and biologically motivated nonlinear rules. The dataset creation process is fully synthetic.The four biological states represented are:1. Cardiac Control2. Cardiac Ischemia-Reperfusion3. Cancer Therapy-Naive4. Cancer Stress-AdaptedThe perturbation conditions represented are:1. WT2. CypD_low3. ANT_low4. ATPsyn_low5. ANT_ATPsyn_low## Preprocessing and CleaningBecause the dataset was generated programmatically, all records are internally consistent and complete by construction. Standard preprocessing mainly involves:- Encoding categorical variables for machine learning- Preserving continuous variables in numeric form- Grouping by state and perturbation for summary analysisNo imputation, de-identification, or normalization of human data was required.## Uses### Intended UsesThis dataset is intended for:- Proof-of-concept mitochondrial modeling- Demonstration of state-dependent stress biology- Exploratory machine learning- Feature importance analysis- Counterfactual perturbation testing- Educational use- Benchmarking synthetic analysis workflows### Not Intended UsesThis dataset is not intended for:- Clinical prediction- Human subject inference- Regulatory decision-making- Validation of therapeutic efficacy- Definitive biological claims without further evidence## Distributional DesignThe dataset was designed so that the modeled states differ in biologically plausible ways.### General distributional intentions- Cardiac Control should show relatively preserved membrane potential, lower calcium stress, lower mtROS, lower pore opening, and higher viability- Cardiac Ischemia-Reperfusion should show lower membrane potential, higher calcium, higher oxidative stress, greater sustained opening, and lower viability- Cancer Therapy-Naive should occupy an intermediate stress region- Cancer Stress-Adapted should retain partial energetic function despite persistent stress and partial pore sensitizationThese are design goals, not measured truths.## Labels### State LabelsState labels indicate the simulated biological context.### Perturbation LabelsPerturbation labels indicate simulated reduction of candidate pore-regulatory mechanisms.### Outcome LabelsOutcome variables such as sustained opening status are derived from upstream variable interactions defined within the simulation rules.## Dataset SizeAdd exact number of rows and columns here after final It may be compared conceptually with published mitochondrial literature, but it is not directly merged from published raw data files. It is literature-benchmarked in design rather than literature-derived row by row.## Legal and Ethical ConsiderationsBecause the dataset is synthetic, it does not contain personal data, patient information, or protected health information. No consent procedures were required. No institutional review board approval was required for dataset generation alone.However, users should avoid representing the dataset as if it were derived from real biological measurements.## Limitations### Synthetic designThe dataset reflects assumptions coded into the simulation.### Simplified biologyImportant mitochondrial factors such as pH, phosphate, adenine nucleotide transport dynamics, mitochondrial morphology, and signaling feedback loops are not fully represented.### Context abstractionThe disease states are simplified abstractions and do not capture full tissue, species, or tumor heterogeneity.### Outcome dependenceDownstream variables are generated from modeled relationships, so predictive analyses partly recover design structure.## MaintenanceIf the dataset is updated, version changes should document:- Any change in variable definitions- Any change in state parameterization- Any added perturbation conditions- Any change in the number of simulated records- Any change in outcome generation logic## Recommended Accompanying DocumentationThis datasheet should be used together with:- `README.md`- `modelcard.md`- The main notebook- Exported figures and summary tables## CitationPetalcorin, M.I.R.
