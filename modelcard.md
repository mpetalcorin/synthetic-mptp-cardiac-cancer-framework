# Model Card## Synthetic Proof-of-Concept Framework for Modeling Mitochondrial Permeability Transition Pore Dynamics## Model Details### Model NameSynthetic mPTP Proof-of-Concept Simulation Framework### Model TypeMechanistically inspired synthetic simulation and analysis framework with supervised machine learning components### VersionVersion 1.0### AuthorMark I.R. Petalcorin### Date31 March 2026## Model SummaryThis model is a synthetic computational framework designed to simulate mitochondrial permeability transition pore behavior across disease-relevant biological states. It combines literature-benchmarked variable design with downstream statistical and machine learning analyses to explore how mitochondrial stress variables influence sustained pore opening, mitochondrial DNA release, ATP-linked function, and viability.The framework is not a mechanistic structural proof of the pore and not a clinical prediction model. It is best understood as an interpretable computational sandbox for exploring biologically plausible relationships across distinct mitochondrial stress contexts.## Intended Use### Primary Intended UsesThe model is intended for:- Hypothesis generation in mitochondrial biology- Comparison of state-dependent mitochondrial stress responses- Simulation-based exploration of candidate pore regulators- Educational demonstration of mPTP-related concepts- Development and testing of interpretable machine learning workflows- Planning future wet-lab experiments### Primary Intended UsersThe intended users include:- Molecular biologists- Mitochondrial biologists- Cell biologists- Computational biologists- Bioinformaticians- Machine learning researchers in biomedicine- Graduate students and educators## FactorsThe model explicitly encodes the following major biological factors:- Mitochondrial membrane potential, ??m- Matrix calcium- Mitochondrial reactive oxygen species, mtROS- Cyclophilin D-associated sensitization- ANT-related function- ATP synthase-associated pore propensity- Disease state- Perturbation conditionThese factors collectively influence modeled outcomes such as sustained pore opening, ATP index, mitochondrial DNA release, and viability fraction.## MetricsThe framework uses descriptive and predictive metrics rather than clinical performance metrics.### Descriptive Outputs- State-level means and standard deviations- Distribution plots across biological states- Grouped perturbation summary values### Machine Learning Outputs- Classification of sustained pore opening status- Permutation feature importance rankings- Regression agreement between predicted and observed viability- Regression agreement between predicted and observed mitochondrial DNA releaseBecause the dataset is synthetic, performance should be interpreted as internal coherence of the framework rather than external biological validation.## Evaluation DataThe evaluation data are synthetic and generated within the notebook itself. Each row corresponds to a simulated single-cell observation. The dataset is structured to represent four biological states and several perturbation conditions. It contains complete records by design and does not rely on patient data or direct experimental measurements.## Training DataThere is no external biological training dataset in the traditional sense. The machine learning components are trained on synthetic records generated from literature-inspired state-dependent distributions and rule-based nonlinear outcome functions. Therefore, the training data reflect assumptions embedded in the simulation design.## Quantitative AnalysesThe classification and regression models are included to test whether the synthetic feature space preserves meaningful structure. Good predictive performance in this context indicates that the synthetic variables are internally coherent and sufficiently informative for the downstream tasks defined in the notebook.This does not demonstrate generalization to external biological data.## Ethical ConsiderationsThis model does not use human subjects data, clinical records, or identifiable biological specimens. It is therefore low risk with respect to privacy and consent concerns. However, there are still important interpretation risks.### Key Ethical Risks- Users may overinterpret synthetic outputs as real biological evidence- Users may mistake internal model coherence for experimental validation- Users may apply the model outside its intended scope### Mitigation Guidance- Always state clearly that the model is synthetic- Do not use outputs for medical decision-making- Treat perturbation findings as hypotheses, not validated interventions- Validate all biological claims experimentally before translational use## Caveats and Limitations### Synthetic NatureThe model is entirely synthetic and literature-benchmarked. It does not use experimentally calibrated measurements from a single biological system.### Structural UncertaintyThe model does not resolve the exact molecular identity of the mitochondrial permeability transition pore. It treats major candidate regulators as biologically motivated inputs rather than settled pore-forming structures.### Biological SimplificationReal mitochondrial biology depends on many additional variables not explicitly encoded here, including pH, phosphate, nucleotide balance, membrane ultrastructure, post-translational modifications, tissue-specific context, and dynamic signaling feedbacks.### Cancer SimplificationCancer is represented here as generalized therapy-naive and stress-adapted states. Real tumors are much more heterogeneous.### Performance InterpretationMachine learning performance reflects synthetic internal consistency, not real-world predictive validity.## Out-of-Scope Use CasesThis model should not be used for:- Clinical diagnosis- Patient prognosis- Therapeutic selection- Real-world risk estimation- Definitive molecular claims about pore architecture- Regulatory submissions## Recommendations for Responsible UseUse the framework to generate mechanistic questions such as:- Which upstream variables most strongly drive sustained pore opening in this simulated setting- Which perturbations appear most promising for follow-up testing- How do acute injury and chronic stress adaptation differ computationally- What measurements should be prioritized in future experimentsDo not use the framework to claim that a specific intervention will work in vivo or in patients without validation.## Future ImprovementsPotential future directions include:- Calibration against isolated mitochondria data- Integration of pH and phosphate dynamics- Time-resolved modeling of transient versus sustained opening- Inclusion of tissue-specific mitochondrial phenotypes- Incorporation of inflammatory signaling downstream of mtDNA release- External benchmarking against real cardiomyocyte and cancer cell datasets## CitationPetalcorin, M.I.R.
